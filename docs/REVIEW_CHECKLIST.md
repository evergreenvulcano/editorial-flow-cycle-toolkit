# Review Checklist

Before merging this branch, confirm:

- The repository does not include the source PDF.
- The vault status is clear.
- AI reading is not yet activated.
- The distinction between source work and content is explicit.
- The Content Desk is defined as absent by default.
- All future AI outputs require status labels.
- No file implies publication, canonization, or derivative permission.
- A colleague can understand the purpose without private context.
