# Product Definition

This repository builds a document-governed, AI-augmented editorial filesystem environment around a protected literary source work.

The product is not the source work.

The product is the environment that allows the source work to be read, mapped, reviewed, protected, and selectively developed without flattening it into content.

## Product promise

To make AI-assisted editorial work possible without allowing the source voice, artistic status, provenance, or material integrity to leak into uncontrolled derivative production.

## Core product object

A protected vault.

## Core user

An editor, author, publisher, researcher, or collaborator who needs to work with a complex literary source while preserving its status.

## Non-goals

- Not a content machine.
- Not a summarizer.
- Not a publishing CMS.
- Not a prompt pack.
- Not a style cloning tool.
- Not a fan wiki.
- Not an archive dump.
