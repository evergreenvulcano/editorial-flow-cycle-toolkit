# Prompt: Extract a Method

Use this prompt to ask an AI assistant to extract a reusable method or pattern from a completed cycle.

This prompt is appropriate when a cycle returned `EXTRACT_ONLY` or when a reusable pattern was identified at Station 6.

---

## Prompt

You are extracting a reusable method or pattern from an editorial cycle.

Your task is to write a reinjection note that describes the pattern or method in general terms.

**You must follow these rules without exception:**

1. Do not name, quote or reproduce source material in the reinjection note without explicit authorization.
2. Describe the pattern in general terms. The note must be readable without access to the source.
3. Do not promote or imply that the source material is a publication, product or public resource.
4. The reinjection note records learning, not content. It does not transfer ownership or rights.
5. If you cannot describe the pattern without referencing the source, state that and ask for authorization before proceeding.

---

**Source cycle reference:**

- Cycle ID: [reference to the cycle that identified the pattern]
- Station where pattern was identified: [typically Station 6]
- Gate decision at that station: [EXTRACT_ONLY / PASS]

---

**Pattern description (from cycle record):**

[Copy the pattern description from the cycle report here.]

---

**Instructions:**

Write a reinjection note using the structure in `templates/reinjection-note.template.md`.

Fill in:

1. A clear description of the pattern or method in general terms.
2. Where this pattern applies (what contexts, material types, situations).
3. How to apply it (concrete and actionable steps or principles).
4. Any conditions under which the pattern should not be applied.
5. The source reference section, stating whether the source is referenced and under what authorization.

The reinjection note must be self-contained. A person who has never seen the source material should be able to use this note.

Do not add claims about the value, quality or importance of the source material.

Do not suggest that this note constitutes publication or public release of any kind.
